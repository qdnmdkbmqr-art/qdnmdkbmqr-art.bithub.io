# 思维流 · MindStream

一个基于思维导图的知识版"抖音"，通过 AI 驱动的多维度思考，帮助你发散思维、头脑风暴、寻找灵感。

## 特性

- **粒子式美学** - 节点像星系般呼吸，视觉优雅
- **AI 实时生成** - 使用 Claude API 动态创建思维分支
- **类抖音体验** - 上下滑动切换问题，流畅探索
- **无限发散** - 点击任意节点继续展开，思维无边界
- **极简设计** - 纯黑背景 + 蓝色粒子，专注思考本身

## 快速开始

1. **打开应用**
   ```bash
   # 直接在浏览器中打开 index.html
   open index.html

   # 或使用本地服务器（推荐）
   python3 -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

2. **配置 API Key**
   - 首次打开会提示输入 Anthropic API Key
   - 获取 API Key: https://console.anthropic.com/
   - API Key 仅保存在本地浏览器，不会上传

3. **开始探索**
   - 上下滑动或滚轮切换问题
   - 点击粒子节点展开更多分支
   - 点击搜索按钮输入自定义问题

## 使用方式

### 随机探索
- 打开应用自动加载随机问题
- 向上滑动获取新的随机问题
- 向下滑动返回之前的问题

### 主动搜索
- 点击左上角搜索图标
- 输入想要探索的问题
- 按回车或点击"探索"按钮

### 深度发散
- 点击任意粒子节点
- AI 自动生成新的思考分支
- 可以无限展开，思维无边界

### 刷新重置
- 点击右上角刷新按钮
- 清空当前思维导图
- 加载新的随机问题

## 技术栈

- **前端框架** - 纯 HTML/CSS/JavaScript（无依赖）
- **渲染引擎** - Canvas 2D API
- **AI 引擎** - Claude 3.5 Sonnet via Anthropic API
- **动画系统** - requestAnimationFrame

## 设计理念

### 思维优先
不给标准答案，只做发散拆解。每个问题都有无数种思考角度，重要的是过程而非结论。

### 极简美学
黑色背景 + 蓝紫色粒子，移除一切干扰，让注意力集中在思维本身。

### 流式体验
像刷短视频一样流畅，但刺激的是大脑而非多巴胺。碎片时间也能深度思考。

### 无限可能
点击任意节点都能继续展开，思维导图永远在生长，没有终点。

## 自定义配置

### 修改视觉风格
编辑 `style.css` 中的 CSS 变量：

```css
:root {
    --bg-dark: #0a0a0f;           /* 背景色 */
    --accent: #6366f1;             /* 强调色 */
    --particle: rgba(99, 102, 241, 0.6);  /* 粒子颜色 */
}
```

### 调整粒子效果
编辑 `mindmap.js` 中的参数：

```javascript
// 节点距离
const distance = 150 + node.level * 100;

// 粒子数量
const particleCount = 20;

// 呼吸速度
const breathe = Math.sin(this.time * 2 + node.level) * 0.2 + 1;
```

### 添加问题库
编辑 `api.js` 中的问题数组：

```javascript
const questions = [
    "你的问题1",
    "你的问题2",
    // ...
];
```

## 常见问题

**Q: API 调用失败怎么办？**
A: 检查 API Key 是否正确，确保网络可以访问 Anthropic API。

**Q: 可以离线使用吗？**
A: 核心界面可以离线查看，但 AI 生成功能需要网络连接。

**Q: 如何保存思维导图？**
A: 目前不支持保存，专注于即时思考和探索体验。

**Q: 可以自定义 AI 提示词吗？**
A: 可以，修改 `api.js` 中的 `generateMindMap` 和 `expandNode` 方法中的 prompt。
